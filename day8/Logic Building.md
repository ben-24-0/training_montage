# Logical Reasoning Roadmap for Placements

> [!NOTE]
> This roadmap covers the majority of Logical Reasoning topics asked in campus placements, aptitude tests, and competitive exams.

---

## 1. Basic Reasoning

### Blood Relations

- Family tree problems
- Paternal and maternal relationships
- Generation-based questions

### Direction Sense

- North, South, East, West
- Distance and displacement
- Shortest path questions

### Ranking & Order

- Position from left/right
- Total number of people
- Relative ranking

### Coding-Decoding

- Letter shifting
- Number substitution
- Symbol-based coding
- Pattern recognition

### Alphabet Test

- Letter positions
- Reverse alphabet
- Word arrangements

### Number & Letter Series

- Missing terms
- Pattern identification
- Sequence completion

---

## 2. Arrangement & Puzzle-Based Reasoning

### Linear Arrangement

- People sitting in a row
- Relative positioning

### Circular Arrangement

- Facing center
- Facing outside
- Clockwise and anti-clockwise relations

### Seating Arrangement

- Multiple constraints
- Mixed arrangements

### Floor-Based Puzzles

- People living on different floors
- Relative floor positions

### Box Arrangement

- Colored boxes
- Stacking problems
- Position constraints

### Scheduling Puzzles

- Dates
- Days
- Time slots
- Event allocation

---

## 3. Analytical Reasoning

### Syllogisms

- All
- Some
- No

Use Venn diagrams for faster solving.

### Statements & Conclusions

- Logical inference
- Validity checking

### Assumptions

- Hidden conditions
- Underlying beliefs

### Cause & Effect

- Event relationships
- Logical chains

### Course of Action

- Problem-solving decisions
- Practical outcomes

### Assertion & Reason

- Statement analysis
- Reason validation

---

## 4. Mathematical Reasoning

### Analogies

- Number analogy

- Word analogy

### Classification (Odd One Out)

### Missing Number

- Matrix patterns
- Numerical relationships

### Mathematical Operations

- Symbol replacement
- Operator-based reasoning

### Clock Problems

- Angle calculations
- Time relationships

### Calendar Problems

- Day finding
- Leap year concepts

### Data Interpretation

- Line Graphs

### Venn Diagrams

- Set relationships
- Logical grouping

# Core Skill Behind All Reasoning Topics

> [!WARNING]
> Most students try to solve reasoning questions mentally. This is the biggest mistake.

Convert words into visual representations.

| Topic | Best Representation |
|---------|-------------------|
| Blood Relations | Family Tree |
| Direction Sense | Coordinate Diagram |
| Ranking | Straight Line |
| Seating Arrangement | Seating Layout |
| Puzzles | Tables & Grids |
| Syllogisms | Venn Diagram |
| Venn Diagrams | Circles |
| Coding-Decoding | Pattern Table |

<details>
<summary>Most Important Placement Strategy</summary>

Whenever you read a reasoning question:

1. Do not solve mentally.
2. Draw the information.
3. Convert statements into:
   - Tables
   - Trees
   - Grids
   - Diagrams
4. Identify the pattern.
5. Solve systematically.

The fastest candidates are usually the ones who visualize the problem rather than memorizing tricks.

</details>
