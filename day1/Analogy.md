# TCS / Infosys Analogy Priority List

## Tier 1 (Asked Very Frequently)

### 1. Part → Whole

```text
Brick : House

Bone : Skeleton

Page : Book
```

Examples:

```text
Bones : Skeleton

Bricks : House
```

---

### 2. Tool → User

```text
Doctor : Stethoscope

Writer : Pen

Surgeon : Scalpel

Carpenter : Saw
```

Examples:

```text
Sculptor : Chisel

Surgeon : Scalpel
```

---

### 3. Young One → Animal

```text
Lamb : Sheep

Cub : Lion

Tadpole : Frog
```

Examples:

```text
Nymph : Cockroach

Lamb : Sheep
```

---

### 4. Category → Member

```text
Snake : Reptile

Toad : Amphibian

Bat : Mammal
```

Examples:

```text
Snake : Reptile

Lizard : Reptile
```

---

### 5. Collective Noun

```text
Wolf : Pack

Teacher : Faculty

Bird : Flock
```

Examples:

```text
Wolves : Pack

Teachers : Faculty
```

---

### 6. Profession → Subject

```text
Cardiologist : Heart

Neurologist : Brain

Astronomy : Stars

Agronomy : Crops
```

Examples:

```text
Cardiologist : Heart

Astronomy : Stars
```

# TCS Shortcut

Whenever you see an Analogy question, check in this order:

```text
1. Synonym?

2. Antonym?

3. Part-Whole?

4. Tool-User?

5. Profession-Subject?

6. Animal-Young One?

7. Category-Member?

8. Collective Noun?

9. Number Pattern?

10. Letter Pattern?
```
